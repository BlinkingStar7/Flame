* <질문 1> 두 번째 숫자를 뒷자리부터 한개씩 떼어내는 것을 어떻게 하면 쉽게 구현할 수 있을까?
* <질문 2> 선기는 다음과 같은 코드를 작성하였다. 맞았을까 틀렸을까?

```c++
	int A, B;
	scanf("%d%d", &A, &B);

	while (B) {
		printf("%d\n", A*(B%10));
		B/=10;
	}

	printf("%d\n", A*B);
```
